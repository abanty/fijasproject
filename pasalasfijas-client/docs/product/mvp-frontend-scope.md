# MVP Frontend Scope

## Objective
Build the frontend shell for a football AI prediction platform.

## Included in this frontend MVP
- Dashboard overview.
- Daily predictions page.
- Match listing page.
- Locked prediction cards for free users.
- Pricing/free vs premium comparison.
- Prediction history mock page.
- Bankroll mock page.
- Service layer prepared for backend integration.

## Freemium behavior
Free users can view 2 predictions per day. Other predictions are visually locked with blur. Backend must enforce this later.

## Premium value
Premium users can view:
- All matches.
- Main pick.
- Alternative picks.
- Combo bets.
- Stake index.
- Full rationale.
- History and performance metrics.

## Betting copy safety
The UI must never promise guaranteed results. Use careful language: analysis, recommendation, confidence, risk, value, NO_BET.
