# MVP Scope

PasalasFijas is a football prediction platform with a freemium subscription model.

## Included in MVP

- Register/login.
- User profile.
- FREE and PREMIUM plans.
- Subscription access model.
- Daily football matches.
- Prediction listing.
- Prediction detail access.
- 2 free full predictions per day.
- Locked prediction previews.
- AI analysis storage.
- Pick storage.
- ComboBet storage.
- Basic bankroll creation/get.
- Admin health route.

## Not included yet

- Real OpenAI generation.
- Real sports data provider.
- Payment provider.
- Automatic match sync.
- Pick settlement jobs.
- Advanced ROI/yield metrics.
- Notifications.
- Scraping.

## MVP goal

Validate product flow before adding costly integrations:

```txt
matches -> analysis -> picks -> freemium access -> frontend display -> result tracking
```
